The Elemental Swordsman

This is my Thesis project for Academy of Art University. The aim of this project was to develop a combo system for a hack and slash type top down adventure game. 
Specific combos let the character perform magic attacks that charge their normal attacks with elements for a small period of time. 
I've designed the pixal art for the character, attacks, and enemies from scratch in Aseprite and brought them to life in unity.
For a more complete description I have also included the GDD for this project
